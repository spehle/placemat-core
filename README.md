# placemat-core

Placemat Core is the backend and infrastructure foundation of **Placemat**,
an open-source inventory and logistics system inspired by real-world intralogistics.

## Goals
- Clean domain model for locations, transport units and inventory
- REST API usable by web frontends and TUIs
- PostgreSQL persistence with Flyway migrations
- Designed for containerized and Kubernetes-based deployments

## Tech Stack
- Java 21
- Spring Boot
- PostgreSQL
- Flyway
- REST API
- ncurses-based TUI client (planned)
